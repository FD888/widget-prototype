# VTB Vita — Tone of Voice виджета

> Статус: **утверждён**
> Последнее обновление: 24 апреля 2026

---

## Суть голоса

Виджет говорит как друг-финансист: разбирается в деньгах, говорит просто и по-человечески, не продаёт. Как сообщение в Telegram от человека, которому доверяешь.

**Тест каждой фразы:** если звучит как менеджер банка — переписать. Если звучит как совет друга — оставить.

---

## Ключевые принципы

| Принцип | Пояснение |
|---------|-----------|
| **Разговорный тон** | Не формальный, не канцелярский. Как пишут в мессенджере — строчными, без точек в конце |
| **Обращение на «ты»** | «можешь», «подключи», «твой» — не «вы» и не «вас» |
| **Без имени** | Не обращаемся по имени — «Ольга, оплатить?» → «платёж по ТТК завтра» |
| **Без эмодзи** | Чистый текст, без визуального шума |
| **Суммы опционально** | Сумма показывается если не чувствительна (кешбэк, выгода). Чувствительные суммы (кредитка, ипотека) — скрываются. См. раздел «Чувствительность данных» |
| **От лица о пользователе** | «можешь возвращать», «платёж просрочен» — не «я могу вернуть» и не безлично |
| **Информативность > краткости** | 2 слова — мало. Лучше чуть длиннее, но с пользой. Если влезает в виджет — норм |

---

## Форматирование

- **Строчные буквы** — «платёж по ТТК завтра», не «Платёж по ТТК завтра»
- **Без точки в конце** — как сообщение в чате
- **Восклицательный знак** — только в приветствиях («доброе утро!»)
- **Вопросительный знак** — только в прямых приглашениях к действию («оплатить?», «стоит глянуть?»)
- **Тире** — для связок («кешбэк на фастфуд — +400 руб/мес»)
- **Числа** — с разделителем-пробелом («4 147 руб», не «4147 руб»)
- **Сокращения** — «руб» вместо «рублей», «мес» вместо «месяцев», «тыс» для тысяч

## Чувствительность данных на виджете

Виджет на главном экране видно всем, кто смотрит на телефон. Это не банковское приложение — здесь нет PIN-защиты. Поэтому фразы на виджете должны быть информативными, но **не раскрывать чувствительные финансовые детали**.

### Правило: что показываем, а что нет

| Категория | Сумма на виджете | Пояснение |
|-----------|-----------------|-----------|
| **Автоплатежи** (ТТК, ЖКХ) | Можно показать | Мелкие регулярные суммы не чувствительны |
| **Кешбэк / выгода** | Показываем | «можешь возвращать 400 руб» — привлекает, не раскрывает баланс |
| **Вклад / накопления** | Показываем выгоду, не баланс | «заработаешь 4 147 руб» — ок, «у тебя 87 340 руб» — нет |
| **ИИС / налоговый вычет** | Можно показать | «до 52 тыс руб из налогов» — оффер, не личные данные |
| **Кредитная карта** | **Без суммы** | Сумма платежа по кредитке — чувствительная информация |
| **Ипотека / кредиты** | **Без суммы** | Крупные суммы по кредитам — не показываем на виджете |
| **Баланс** | **Никогда** | Текущий баланс счёта — только в приложении с PIN |

### Шаблоны с учётом чувствительности

**Нечувствительные платежи** (автоплатежи — ТТК, ЖКХ):
- «платёж по ТТК завтра, 690 руб» — сумма ок
- «сегодня платёж по ЖКХ, 6 800 руб» — сумма ок

**Чувствительные платежи** (кредитка, ипотека, кредиты):
- «платёж по кредитке завтра» — без суммы
- «кредитная карта просрочена, оплатить?» — без суммы
- «ипотека скоро спишется» — без суммы

**Выгода** — всегда с числом выгоды, но без раскрытия баланса:
- «можешь возвращать 400 руб в месяц кешбэком» — ок
- «заработаешь 4 147 руб за 3 мес на вкладе» — ок (выгода, не баланс)
- ~~«твой остаток 87 340 руб»~~ — нет, раскрывает баланс

### Критерий для разработчика

```
Если payment_type in ('credit_card', 'loan') → не показываем сумму на виджете
Если payment_type == 'autopayment' → можно показать сумму
Если оффер ВЫГОДА → показываем выгоду (доход), но не баланс пользователя
```

---

Фоновые фразы без информационного посыла. Виджет живой, не давит. Средняя длина — 3-5 слов.

### По времени суток

| Период | Фразы |
|--------|-------|
| Утро (6-10) | «доброе утро!» · «с добрым утром» · «хорошего дня впереди» |
| День (11-16) | «на связи, если что» · «чем могу помочь?» · «всё под контролем» · «привет, я тут» · «как дела?» |
| Вечер (17-21) | «добрый вечер!» · «как прошёл день?» · «на связи» |
| Ночь (22-5) | «я на связи, если что» · «тихо сегодня» · «всё под контролем» |

## Тип 1 — Напоминания

Своевременные, конкретные, полезные. Только когда есть реальное событие — это сервис, не реклама.

### Принципы

- Всегда содержит **название платежа** и **сумму**
- Срочность передаётся формулировкой, не восклицаниями
- Вопрос «оплатить?» — только для просроченных

### Шаблоны по срочности

| Состояние | Чувствительный (кредитка, ипотека) | Нечувствительный (автоплатеж) |
|-----------|-------------------------------------|-------------------------------|
| Просрочен | «платёж по кредитке просрочен, оплатить?» | «платёж по ТТК просрочен, 690 руб — оплатить?» |
| Сегодня | «сегодня платёж по кредитке» | «сегодня платёж по ТТК, 690 руб» |
| Завтра | «платёж по кредитке завтра» | «платёж по ТТК завтра, 690 руб» |
| Через 2-3 дня | «платёж по кредитке через 2 дня» | «платёж по ТТК через 2 дня, 690 руб» |

### Пул фраз (альтернативы для ротации)

**Просрочен:**
- «платёж по {название} просрочен, оплатить?»
- «{название} просрочен{, сумма} — оплатить?» *(сумма только для autopayment)*

**Сегодня:**
- «сегодня платёж по {название}{, сумма}»
- «{название} списывается сегодня{, сумма}»

**Завтра:**
- «платёж по {название} завтра{, сумма}»
- «завтра спишется {название}{, сумма}»

**Через 2-3 дня:**
- «платёж по {название} через {N} дня{, сумма}»
- «скоро {название} — через {N} дня{, сумма}»

> **{, сумма}** — подставляется только для нечувствительных платежей (autopayment).
> Для credit_card и loan — пропускется.

## Тип 2 — ВЫГОДА

Тег ВЫГОДА + числовая выгода + 1 действие. Без тройки «тег + цифра + действие» — не показывать.

### Принципы

- Всегда содержит **конкретную сумму выгоды**
- Формулировка показывает, что пользователь **может сделать**, а не что банк ему **предлагает**
- Тон — «кстати, ты можешь...», а не «купите наш продукт»

### Шаблоны по офферу

| Оффер | Шаблон виджета | Пример |
|-------|---------------|--------|
| **Кешбэк** | «можешь возвращать {сумма} руб в месяц кешбэком за {категория}» | «можешь возвращать ~400 руб в месяц кешбэком за фастфуд» |
| **Вклад 3 мес** | «если перевести свободные деньги на вклад, за 3 мес заработаешь {сумма} руб» | «если перевести свободные деньги на вклад, за 3 мес заработаешь 4 147 руб» |
| **ИИС** | «ИИС вернёт до {сумма} руб из налогов — стоит глянуть» | «ИИС вернёт до 52 тыс руб из налогов — стоит глянуть» |
| **Автоплатёж** | «автоплатёж позволит не держать в голове плату за {платежи}» | «автоплатёж позволит не держать в голове плату за ТТК и ЖКХ» |
| **Prime** | «с праймом переводы без комиссии — {сумма} руб экономии в месяц» | «с праймом переводы без комиссии — ~200 руб экономии в месяц» |
| **Округление** | «округляй покупки — незаметно накопишь ~{сумма} руб в месяц» | «округляй покупки — незаметно накопишь ~50 руб в месяц» |

### Пул фраз (альтернативы для ротации)

**Кешбэк:**
- «можешь возвращать {сумма} руб в месяц кешбэком за {категория}»
- «кешбэк на {категория} вернёт тебе ~{сумма} руб в месяц»

**Вклад:**
- «если перевести свободные деньги на вклад, за 3 мес заработаешь {сумма} руб»
- «можно заработать {сумма} руб за 3 мес, если перевести остаток на вклад»

**ИИС:**
- «ИИС вернёт до {сумма} руб из налогов — стоит глянуть»
- «можно вернуть до {сумма} руб из налогов через ИИС»

**Автоплатёж:**
- «автоплатёж позволит не держать в голове плату за {платежи}»
- «{платежи} платились бы сами с автоплатежом»

## Тип 3 — Аналитика (спроектирована)

Только на виджете, без баннера. Ценна сама по себе — пользователь получает знание, без призыва что-то купить.

| Триггер | Шаблон |
|---------|--------|
| Конец месяца | «{месяц} закрыт — посмотреть куда ушло?» |
| Конец месяца + аномалия | «{месяц}: потратил на {разница} руб больше, чем в {предыдущий месяц}» |
| Крупная трата < 24 ч | «потратил {сумма} руб — осталось {процент}% бюджета на неделю» |

---

## Что НЕ говорить

| Фраза / паттерн | Почему нет |
|-----------------|-----------|
| «Снова перевести Маше?» | Навязчиво — пользователь сам помнит |
| «Пятничный перевод?» | Пустая фраза, нет полезной информации |
| «Зарплата пришла — куда отложить?» | Абстрактно, нет персональной цифры |
| «Деньги лежат без дела — вклад 18%» | Звучит как реклама |
| «Возвращай деньги с каждой покупки» | Слоган, не совет |
| ВЫГОДА без числа | Нарушает главный принцип — конкретная выгода |
| «Оформите», «Подключите», «Вам доступно» | Канцелярщина, банковский тон |
| «Уважаемый», «Дорогой» | Фальшивая вежливость |
| Капслок | КРИЧИТ — виджет не должен давить |

## Приоритет между типами

При каждом обновлении виджета выбирается один текст:

```
Напоминание (если есть актуальный платёж)
    ↓ нет
ВЫГОДА (по логике, с учётом ограничений частоты)
    ↓ нет
Нейтральная (из пула по времени суток)
```
