# VTB Vita — Design Guidebook
### На основе анализа скриншотов приложения ВТБ Онлайн

---

## 1. Цветовая палитра

### Основные цвета (Dark Theme — единственная тема)

| Роль | HEX | Где используется |
|------|-----|-----------------|
| **Background Primary** | `#0D0D0D` / `#111111` | Основной фон экранов (глубокий чёрный) |
| **Background Surface** | `#1A1A1A` | Карточки, поля ввода, блоки-секции |
| **Background Surface Alt** | `#222222` | Слегка поднятые элементы, bottom sheet |
| **Background Elevated** | `#2A2A2A` | Chip-кнопки (+500₽, +1000₽), теги |
| **Text Primary** | `#FFFFFF` | Заголовки, суммы, имена |
| **Text Secondary** | `#8A8A8A` | Подписи («Откуда», «Кому», «Сумма»), лейблы полей |
| **Text Hint** | `#5C5C5C` | Placeholder в полях ввода |
| **Accent Blue** | `#4A7FFF` / `#3D7BFF` | Ссылки («Настроить», «Подробнее»), CTA-акценты |
| **Accent Green** | `#34C759` | Успех (иконка чекмарка), «Комиссия 0₽», статус выполнено |
| **Warning/Alert** | `#A0522D` (коричневый) | Предупреждение на экране подтверждения |
| **VTB Brand Blue** | `#0061D5` | Лого, иконка Мир-карты (синяя), кнопка выбора банка |

### Главный экран (отличается от остальных)

На главном экране используется **градиентный фон**: синий → фиолетовый → тёмно-синий.  
Это единственный экран с цветным фоном — все транзакционные экраны строго тёмные.

```
Background gradient: #1A3A8C → #2D1B69 → #0D1B4B (примерно)
```

## 2. Типографика

### Иерархия шрифтов

ВТБ использует системный шрифт Android (Roboto) с чёткой иерархией весов.

| Уровень | Размер | Вес | Применение |
|---------|--------|-----|------------|
| **Hero** | 32–36sp | Bold (700) | Сумма на главном экране («5,00 ₽») |
| **H1** | 24–28sp | Bold (700) | Заголовок секции («Перевод», «Получатель», «Детали») |
| **H2** | 18–20sp | SemiBold (600) | Заголовок экрана в AppBar («Перевести», «Подтверждение») |
| **H3** | 16–18sp | Bold (700) | Название оператора, имя получателя |
| **Body** | 14–15sp | Regular (400) | Основной контент, телефон, сумма в полях |
| **Caption** | 12–13sp | Regular (400) | Лейблы полей («Откуда», «Кому»), вторичный текст |
| **Label** | 11–12sp | Regular (400) | Подписи в Bottom Nav, мелкие пояснения |

### Паттерны отображения данных

- **Сумма:** крупно, белым, `Bold`. Знак ₽ — того же размера или чуть меньше, рядом
- **Лейбл поля** (Caption, серый) стоит НАД значением, не внутри
- **Имя получателя** — белый Bold, телефон под ним — серый Regular
- **Номер карты** — показывается как `•• 3178` или `...3178`, не полностью

## 3. Компоненты UI

### 3.1 AppBar / Navigation Header

```
┌─────────────────────────────────────┐
│  ←    Заголовок экрана          (i) │
└─────────────────────────────────────┘
```

- Тёмный фон (`#0D0D0D`), сливается с экраном — нет чёткой границы
- Стрелка назад (`←`) — слева, белый цвет, без фона
- Заголовок — по центру, 18–20sp SemiBold, белый
- Иконка информации `(i)` — справа, только если нужна (не везде)
- **Нет тени, нет border** — AppBar невидим как отдельный элемент

### 3.2 Карточки / Info Cards

```
┌─────────────────────────────────────┐
│  Откуда  ▼                     [МИР]│
│  Мастер-счет в рублях  ••3178        │
│  5 ₽                                │
└─────────────────────────────────────┘
```

- Background: `#1A1A1A` (поверхность)
- Радиус скругления: **16dp**
- Padding: ~16dp со всех сторон
- Лейбл (Caption, серый) сверху, значение (белый) снизу
- Маленький `▼` после лейбла — признак кликабельности/выбора
- Логотип банка/карты — справа, в квадратной плашке (~40×40dp)

### 3.3 Поля ввода

```
┌─────────────────────────────────────┐
│                                     │  ← пустое поле, hint серый
└─────────────────────────────────────┘
```

- Background: `#1A1A1A`
- Радиус: **12–16dp**
- Border: отсутствует в обычном состоянии (только фон выделяет поле)
- Placeholder: серый, Caption
- Текст ввода: белый, Body
- **Нет underline**, нет Material outline — плоский тёмный прямоугольник

### 3.4 Primary Button (CTA)

```
┌─────────────────────────────────────┐
│           Продолжить                │
└─────────────────────────────────────┘
```

- Background: `#FFFFFF` (белый!)
- Text: `#000000` (чёрный), 16–17sp SemiBold
- Радиус: **28–32dp** (pill / полностью скруглённый)
- Высота: ~56dp
- Ширина: full-width (с margin 16dp с каждой стороны)
- **Расположение:** всегда внизу экрана, "прибито" к bottom
- Варианты текста: «Продолжить», «Перевести», «Оплатить ₽», «На главную»

### 3.5 Chip / Quick Amount Buttons

```
┌──────────┐  ┌──────────┐  ┌──────────┐
│ +500 ₽   │  │+1 000 ₽  │  │   Все    │
└──────────┘  └──────────┘  └──────────┘
```

- Background: `#2A2A2A`
- Text: белый, 14sp Regular
- Радиус: **20dp** (pill)
- Padding: ~12dp × 20dp
- Нет border в неактивном состоянии

### 3.6 Chip / Bank Selector (горизонтальный скролл)

```
┌──────────────┐  ┌──────────────┐  ┌──────────────────┐
│ ✦ Другой     │  │ WB  WB       │  │[озон] Озон  ✓    │
└──────────────┘  └──────────────┘  └──────────────────┘
```

- Горизонтальный скролл без скроллбара
- Неактивный: `#2A2A2A`, текст белый, нет border
- **Активный (выбранный):** border `#FFFFFF` 1dp, фон без изменений
- Иконка банка слева, название справа

### 3.7 Контакты / List Items

```
┌───────────────────────────────────────┐
│  [АШ]  Агния Юнарм 10шк               │
│        +7 963 164-40-59               │
└───────────────────────────────────────┘
```

- Аватар-инициалы: круг ~40dp, Background `#2A2A2A`, текст белый 14sp
- Имя: белый, 15sp Regular
- Телефон: серый (`#8A8A8A`), 13sp Regular
- Нет разделителей (dividers) — элементы разделяются только отступами
- Секционный заголовок («Контакты», «А», «Б»): серый Caption, выровнен по левому краю

### 3.8 Toggle / Switch

- Стандартный Material switch
- Off: серый трек
- On: синий трек (`#4A7FFF`)

### 3.9 Поле ввода телефона (специфичный паттерн)

```
  🇷🇺  +7 000 000-00-00
```

- Нет рамки, нет карточки — просто текст с флагом
- Размер: 20–24sp, белый
- Placeholder серый
- Флаг как эмодзи или Image

### 3.10 Warning Block

```
┌─────────────────────────────────────────────────────┐
│  Проверьте ФИО и номер получателя.                  │
│  После перевода банк не сможет вернуть деньги.      │
└─────────────────────────────────────────────────────┘
```

- Background: коричневый/янтарный `#3D2010` или `#4A2C0A`
- Text: белый/кремовый, 14sp
- Радиус: **12dp**
- Нет иконки — чистый текстовый блок

### 3.11 Success Screen

```
        ┌────────────────┐
        │   [✓ в круге]  │   ← синий круг + зелёный значок поверх
        └────────────────┘
     Перевод выполнен
     По номеру телефона
          10 ₽
```

- Иконка: тёмно-синий круг (~80dp) с зелёным чекмарком поверх
- Заголовок: белый Bold 20sp
- Подзаголовок: серый Regular
- Сумма: белый Bold 28sp
- После этого блока — список дополнительных действий (Получить чек, Создать шаблон)

### 3.12 Дополнительные действия (Action List)

```
  [📄]  Получить чек
  [+]   Создать шаблон
  [↩]   Как вернуть деньги
```

- Фон секции: `#1A1A1A`
- Иконка слева: outline, белая, ~24dp
- Текст: белый, 15sp Regular
- Нет стрелки вправо — минимализм
- Разделены тонкими линиями `#2A2A2A` или просто отступами

## 4. Структура экранов / Layout Patterns

### 4.1 Транзакционный экран (перевод, пополнение)

```
┌─────────────────────────────────┐
│ AppBar (← Заголовок)            │
├─────────────────────────────────┤
│                                 │
│  [Card: Откуда]                 │
│                                 │
│  [Card: Кому]                   │
│                                 │
│  [Chip Scroll: банки]           │
│                                 │
│  [Label: Сумма]                 │
│  [Input Field]                  │
│  [Quick Amounts: +500 +1000]    │
│                                 │
│  [Label: Сообщение]             │
│  [Input Field]                  │
│  [Emoji Chips]                  │
│                                 │
│                    ↕ скролл     │
├─────────────────────────────────┤
│  [Primary Button: Продолжить]   │
└─────────────────────────────────┘
```

**Правила layout:**
- Горизонтальные отступы: **16dp** от края экрана
- Вертикальные gap между карточками: **12–16dp**
- CTA-кнопка: `margin 16dp`, `padding bottom` = safe area + 16dp
- Скролл — всё выше кнопки, кнопка зафиксирована

### 4.2 Экран подтверждения

```
┌─────────────────────────────────┐
│ AppBar (← Подтверждение)        │
├─────────────────────────────────┤
│                                 │
│  Перевод          ← H1          │
│  Сумма            ← Caption     │
│  10 ₽             ← Body        │
│                                 │
│  Получатель       ← H1          │
│  ФИО / Телефон                  │
│                                 │
│  [Warning Block]                │
│                                 │
│  Банк получателя  ← Caption     │
│  Сбербанк         ← Body        │
│                                 │
│  Детали операции  ← H1          │
│  ...                            │
├─────────────────────────────────┤
│  [Primary Button: Перевести]    │
└─────────────────────────────────┘
```

- Секции разделяются заголовками H1, **нет карточек** — плоский список
- Каждый параметр: Caption (серый) → значение (белый) с gap ~2dp

### 4.3 Bottom Sheet (выбор счёта)

- Выезжает снизу поверх текущего экрана
- Тёмный overlay за ним
- Background: `#1C1C1E` (чуть светлее основного)
- Заголовок вверху: Bold 18sp
- Handle (ручка): тонкая серая черта вверху, ~32dp × 4dp, радиус 2dp

## 5. Иконография

### Стиль иконок

- **Outline** (не filled) — все иконки контурные, белые
- Stroke width: ~1.5–2dp
- Размер: 24dp стандарт, 20dp в компактных местах

### Специфичные иконки ВТБ

| Иконка | Где | Описание |
|--------|-----|----------|
| Звёздочка ✦ (ВТБ Vita?) | Аватар пользователя | AI-маркер поверх иконки профиля |
| Щит с галочкой | AppBar главной | Верификация / безопасность |
| Микрофон | AppBar главной | Голосовой ввод |
| Лупа | AppBar главной | Поиск |
| QR-сетка | Раздел платежей | QR-оплата |
| СБП (разноцветная) | Список способов | Система быстрых платежей |

## 6. Навигация

### Bottom Navigation Bar

```
┌────────────────────────────────────────────┐
│ [🏠]      [↗]       [⊞]      [🕐]    [💬] │
│Главная  Платежи  Продукты  История   Чат   │
└────────────────────────────────────────────┘
```

- Background: `#1A1A1A`, скруглённые верхние углы ~16dp
- Активная иконка + подпись: **белые**
- Неактивные: серые `#8A8A8A`
- Нет underline или pill-индикатора — только цвет
- Высота: ~56–60dp

## 7. Motion / Анимации

- **Переходы между экранами:** slide from right (стандарт Android)
- **Bottom Sheet:** slide from bottom, `300ms`, ease-out
- **Chip selection:** мгновенно, border появляется без анимации
- **Success screen:** иконка появляется с небольшим scale-in
- Никаких сложных анимаций — скорость и минимализм

## 8. Правила применения для VTB Vita Widget

### Виджет главного экрана Android

Виджет живёт на рабочем столе, поэтому:

1. **Фон виджета:** `#1A1A1A` с радиусом `16dp` — как карточка в приложении
2. **Акцентный цвет:** синий `#0061D5` или `#4A7FFF` для кнопок действий
3. **Шрифт:** Roboto, Bold для суммы/результата, Regular для подписей
4. **Кнопки действий внутри виджета:** белые pill-кнопки (как CTA в приложении)
5. **Иконки:** outline, белые, 24dp

### InputActivity / TransferFlowActivity (полноэкранный UI)

Следовать паттернам раздела 4 строго:
- Dark background `#0D0D0D`
- Карточки `#1A1A1A`, радиус `16dp`
- CTA внизу — белая pill-кнопка, full-width
- Никаких светлых фонов, никаких цветных кнопок (кроме акцентов-ссылок)

### Успех операции

Использовать паттерн раздела 3.11:
- Синий круг + зелёный чекмарк
- Белый Bold заголовок
- Серый подзаголовок
- Итоговая сумма Hero-размером

## 9. Чего НЕ делать (Anti-patterns)

| Нельзя | Почему |
|--------|--------|
| Светлый/белый фон на экранах | ВТБ — строго dark mode |
| Material filled buttons синего цвета | CTA всегда белый в тёмной теме |
| Underline в текстовых полях | Поля — плоские тёмные прямоугольники |
| Dividers между элементами списка | Разделение — только отступами |
| Цветной AppBar | AppBar сливается с фоном |
| Capslock в заголовках | Заголовки — обычный case |
| Маленькие радиусы (<8dp) | ВТБ использует мягкие скругления 12–32dp |
